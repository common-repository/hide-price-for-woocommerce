Hide Price for woocommerce will hide price until login.

== Description ==

Hide Price for WooCommerce is an add-on plugin for WooCommerce, By using this plugin you can set the condition to display Price and Add to cart button.

== Installation ==

* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin;
* Navigate to Plugin Settings and fill settings.
